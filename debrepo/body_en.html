Instructions

Run the following commands as root:

echo "deb http://debrepo.appspot.com/ stable main" > /etc/apt/sources.list.d/debrepo.list

apt-get update

iExpresso Setup

Run the following command as root:

apt-get install iexpresso

Start iExpresso: <Main Menu>-<Intenet>-<iExpresso>. Inform your Expresso username and password and click [OK].

Now configure Thunderbird, or your email client, to access the IMAP server at localhost with your Expresso credentials.
